# Gega Movie

Gega Movie is a project made with tailwind.

# Dependencies Used In The Project
- Tailwind
- HTML5

# Installation
Follow the steps in the tailwind documentation for tailwind installation:
https://tailwindcss.com/docs/installation

## Directory Structure

| Path.                                           | Description                                                     |
|-------------------------------------------------|-----------------------------------------------------------------|
| `/public/`                                         | styles.css and images                                |
| `/public/index.html/`                              |  index.html                    |
| `/input.css/`                             |  Base Layer.                                        |
| `/tailwind.config.js`                               | - Tailwind config settings.                                                        |

# Deploy
This project deployed by using Vercel.

- https://gegamovie.vercel.app/
